- Purpose: Windeath admin portal (Next.js App Router) that lets ops/community teams review memorial applications, monitor activity, and jump to external observability tools.
- Core structure: `app/components/dashboard/AdminOverview.tsx` renders the main dashboard UI; `/admin/applications` hosts the entire memorial moderation workflow (Zustand `useMemorialStore`, `ApplicationList`, `ApplicationDetail`); `/app/api/applications/*` proxy all CRUD calls to `MEMORIAL_API_URL`; `middleware.ts` guards `/admin/**` by checking `auth_token` and redirecting to `/admin/dashboard/auth/login`.
- Supporting pieces: `/app/api/auth/*` for JWT verification, `/app/api/health` for status, observability shortcut links target external Grafana/Kiali/Prometheus/Kafka pages, and shared config lives in `public/`, `next.config.ts`, `tsconfig.json`, `eslint.config.mjs`.
